# Walmart Sales Analytics

## Project Overview
This project performs Exploratory Data Analysis (EDA) on Walmart Sales data using Python and SQL. It focuses on data cleaning, visualization, and solving business problems using advanced SQL queries.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL
- Jupyter Notebook
